# Customer Segmentation & RFM Analysis

## Overview
Advanced customer segmentation using RFM (Recency, Frequency, Monetary) analysis and K-Means clustering to identify high-value customer segments and optimize marketing strategies.

## Features
- **RFM Analysis**: Segment customers by recency, frequency, and monetary value
- **K-Means Clustering**: Advanced customer segmentation
- **Customer Lifetime Value**: Calculate CLV for each segment
- **Churn Risk Assessment**: Identify at-risk customers
- **Segment Profiles**: Detailed characteristics of each segment

## Project Structure
```
├── data/              # Customer transaction data
├── src/               # Analysis modules
├── notebooks/         # Exploratory analysis notebooks
├── requirements.txt   # Dependencies
└── main.py           # Main analysis script
```

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```

## Key Metrics
- Recency (days since last purchase)
- Frequency (number of purchases)
- Monetary (total spend)
- Customer Lifetime Value
- Segment Size and Characteristics
